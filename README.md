# 🎬 Movie Recommendation System

A beginner-friendly Python project that recommends movies based on the user's favorite genre.

## 🚀 Features

- 👤 User Name Input
- 🎭 Genre Selection
- 🎬 Movie Recommendations
- 🔁 Uses Python Lists
- 🔄 Uses Loops
- 🧠 Uses if-elif-else Decision Making
- 💻 Console Based Interface

## 🛠 Technologies Used

- Python 3
- PyCharm
- Git
- GitHub

## 📂 Project Structure

```
MovieRecommendationSystem
│── main.py
│── README.md
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/itz-rishi/MovieRecommendationSystem.git
```

2. Open the project in PyCharm

3. Run

```bash
python main.py
```
